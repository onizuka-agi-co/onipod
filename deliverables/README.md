# Deliverables

このディレクトリには、Podmanの監視および管理機能を提供する成果物が含まれています。

## 含まれるもの

- `podman_manager.sh`: Podmanコンテナの監視および管理スクリプト
- `podman_manager_documentation.md`: 上記スクリプトの利用ドキュメント

## 概要

これらの成果物は、Strategyドキュメントで示された通り、「コンテナ管理機能の追加 (Podman監視・管理スクリプト)」の実現を目指しています。

`podman_manager.sh`スクリプトは以下のような機能を提供します：
- コンテナの一覧表示
- 特定コンテナの状態確認
- コンテナの開始、停止、再起動
- コンテナログの表示
- 全コンテナの継続的モニタリング
- 全コンテナのヘルスチェック

## 使用方法

詳細な使用方法については、各ファイルのドキュメントを参照してください：
- `podman_manager_documentation.md`

## フォーカス

現在の開発フォーカスは、コンテナ環境の運用効率化とモニタリングの自動化にあります。今後の拡張として、より高度な監視機能、通知機能などを追加していく予定です。
