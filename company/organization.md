# 👥 Organization

## 組織図

```
        ┌─────────────┐
        │     CEO     │  戦略・組織運営
        │  (30分毎)   │  会議ルール改善
        └──────┬──────┘
               │
        ┌──────┴──────┐
        │     CTO     │  技術実務
        │  (30分毎)   │  コード品質管理
        └─────────────┘
```

## 役割定義

### CEO (Chief Executive Officer)
- **責務**: 組織全体の方向性決定
- **権限**: 会議ルール、戦略、フォーカスの変更
- **頻度**: 毎時30分

### CTO (Chief Technology Officer)
- **責務**: 技術的な実務遂行
- **権限**: コード変更、ドキュメント更新の提案
- **頻度**: 毎時00分
